# 整えるダイエットサポート | ととのサロン

ととのサロンの受講生向けダイエットサポートツール。スマホのブラウザで開くだけで使えます。

**公開URL**: https://tomominblog.github.io/totono-diet-support/

## 機能

- 🌸 **体重チェック** — 身長・体重・年齢からBMIと適正体重・美容体重・シンデレラ体重・年齢別の健康範囲を判定し、判定に応じた食事アドバイスを表示
- 📈 **体重記録** — 毎日の体重を記録して推移グラフで確認（目標体重ライン付き）
- 🍽️ **カロリー計算** — 基礎代謝（Mifflin-St Jeor式）から1日の目安カロリーとPFCバランスを計算。基礎代謝を下回らないガード付き
- 💡 **食べ方のコツ** — 「整える食べ方」のコツと食事アイデア集
- 📝 **食事メモ** — 毎日の食事と気分を記録

## 技術メモ

- 単一 `index.html`（バニラHTML/CSS/JS、ビルド不要）
- データはすべて端末の localStorage に保存（サーバー送信なし）
  - `totono-diet-weight-v1` — 体重記録
  - `totono-diet-memo-v1` — 食事メモ
  - `totono-diet-profile-v1` — プロフィール・目標体重
- 年齢別目標BMI範囲は厚生労働省「日本人の食事摂取基準（2020年版）」、BMI分類は日本肥満学会に準拠
- manifest.json + アイコンで「ホーム画面に追加」に対応
